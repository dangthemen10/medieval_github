# medieval_github

## Cách chạy extension

- install package

```bash
npm i
```

- Build

```bash
npm run build
```

- Mở Chrome và truy cập chrome://extensions/.
- Bật chế độ Developer mode (góc phải trên).
- Chọn Load unpacked và chọn thư mục my-chrome-extension.
- Truy cập trang github để kiểm tra.
